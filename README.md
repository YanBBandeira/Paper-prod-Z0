# Forward $Z^0 \rightarrow \mu^+ \mu^-$ production in $pp$ collision using the CDSM framework 

## Phenomenological point of view

In this project we will use the following kinematic cuts: rapidity range $2.0<\eta<4.5$, dimuons transverse momentum $p_T > 20$ GeV, invariant mass $60 < M_{\mu\mu} < 120 \ \ \text{GeV/c}^{2}$ at $\sqrt{s}=13$ TeV.
